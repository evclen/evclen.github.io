This quilt I made for a young girl who loves unicorns. The unicorn is a pattern by Elizabeth Hartman https:\/\/elizabethhartman.com\/<\/a> and I added some starts and some squares of unicorn fabric to make it bigger along with a nice border of rainbow stary fabric. I found a lovely line of unicorn fabric called Believe by Kim Schaefer and found fabrics from my stash and from the shop to go with them. The finished size is around a meter square. It’s not possible to tell from the photo but the horn is done with glittery fabric!<\/p>\n\n\n\n

I quilted this quilt using stich in the ditch around the unicorn and the star boarder then added some machined stars in navy on the background for extra security. The backing fabric is a super vibrant flowery print which I really didn’t buy enough of so had to do a few joins to make it work but it was definately worth it.<\/p>\n\n\n\n
