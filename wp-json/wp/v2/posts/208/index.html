I decided to start on an English paper piecing project to have some hand sewing to do whilst travelling or watching the television. I wanted to do large hexagons which would all fit together nicely, but not all be the same design. Different designs would be more interesting for me to sew, and I could learn more by doing different designs. After ordering some triangular graph paper, I set to work on some designs.<\/p>\n\n\n\n
